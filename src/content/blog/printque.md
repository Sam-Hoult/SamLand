[**PrintQue**](https://github.com/PrintQue-cat/PrintQue) is an open source project for 3D printer fleet management and automation. If you're running more than one printer (or a farm) you know how quickly things get messy: queue management, job routing, status, and basic automation. PrintQue is aimed at filling that gap in a way that stays open, hackable, and community-driven.

It's the kind of tool I wished existed when I was deep in making and printing, so I'm building it with Zac Hartley. Fleet management, automation, and visibility without locking you into a closed ecosystem. More details and links as the project matures.
